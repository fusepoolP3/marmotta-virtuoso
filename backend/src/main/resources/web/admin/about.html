Storage Backend: Virtuoso

This module provides an Apache Marmotta storage backend based on the Virtuoso Universal Store. Virtuoso is an innovative enterprise grade multi-model data server for agile enterprises & individuals. It delivers an unrivaled platform agnostic solution for data management, access, and integration.
